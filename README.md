# tesi-gis

## Getting started

- Clone the repo

```sh
git clone git@github.com:oPauloChaves/tesi-gis.git
```

- Install add deps

```sh
cd tesi-gis
# with Yarn
yarn install
# or using npm
npm install
```

- Run it in dev mode

```sh
cd tesi-gis
cp .env.example .env # after copying config your env vars
yarn run dev
# or with npm
npm run dev
```
